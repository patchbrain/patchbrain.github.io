hexo博客一些常用功能

hexo s : 启动博客
hexo n : 创建一篇文章
hexo g(generate) : 生成静态文件
hexo d(deploy) : 部署博客到git

一般发布或者修改博客后的操作： ： 清楚缓存——>生成静态文件——>启动服务器——>部署
